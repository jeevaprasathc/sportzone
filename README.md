# 🏆 SportZone - Sports Kits Store

SportZone is a full-stack Sports Kits E-Commerce Website developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB. The platform provides an intuitive shopping experience where users can browse sports products, search for items, add products to their cart, and place orders through a responsive and user-friendly interface.

---

## 📌 Project Overview

SportZone aims to provide a modern online platform for purchasing sports equipment and accessories. The website is designed with a clean user interface, responsive layouts, and efficient backend functionality to ensure a seamless shopping experience.

## 🚀 Installation and Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/sportzone.git
```

### 2️⃣ Navigate to Project Directory

```bash
cd sportzone
```

### 3️⃣ Install Dependencies

```bash
cd backend
npm install
```

### 4️⃣ Configure Environment Variables

Create a `.env` file inside the backend directory and add:

```env
MONGODB_URI=your_mongodb_connection_string
PORT=5000
```

### 5️⃣ Start the Server

```bash
npm start
```

### 6️⃣ Open the Frontend

Open:

```text
frontend/index.html
```

in your browser.
